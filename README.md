# PingAnalyserPLT
version: 1.0.0

PingAnalyserPLT is a network pentesting tool that analysis ping response times with showing simple graph.

## Installation
```bash
pip install -r requirements.txt
```
## Sample usage

```bash
python ping_analyser_plt.py -u <any_website> -t <time>
```
